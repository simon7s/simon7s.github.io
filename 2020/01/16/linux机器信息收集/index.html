linux机器信息收集

基本情况

ifconfig
route 查看路由
whoami或id 查看当前用户id
id test 查看test用户信息
w 查看当前系统活动用户
lastlog 查看用户登录日志
grep MemTotal/porc/meminfo 查看系统内存总量

linux下所有的东西都是文件（以文件形式存在）

proc（系统硬件的东西）

用户列表

查看用户列表 cat/etc/passwd
查看用户组 cat/etc/group
列出超级用户 grep -v -E “^#” /etc/passwd|awk -F: ‘$3 == 0 {print $1}’
列出超级用户 awk -F: ‘($3 == “0”) {print}’ /etc/passwd

awk审计工具

大于1000，后来新增的机器

端口列表

netstat -an
netstat -antlp 查看tcp连接

查看pid（进程），需要啊root权限才可以看

系统类型和内核版本

查看系统名称 cat/etc/issue
查看系统名称、版本号 cat/etc/lsb-release
查看内核信息 uname -a
查看内核信息 ls /root | grep vmlinuz

前两个查看出来是发行版本

uname -a查看出是内核版本

进程信息

ps aux命令下的参数以及其代表意思

PID %CPU %MEM VSZ RSS
进程号 cpu占用 内存使用率 总虚拟内存大小, 常驻集大小
TTY STAT START TMIE COMMAND
进程状态 行程开始时间 执行时间 所执行的指令的名称和参数

上面查看的java号（pid），可以查看它进程

单独使用ps （ps下面有很多命令用法，可以ps –help查看）

安装的软件包和服务

查看安装的软件包 dpkg -l
查看安装的软件包 rpm -qa
查看系统存在的服务 cat /etc/service

service 命令

服务配置

linux下的服务配置都带d

痕迹清理（后面的模块）

btmp

wtmp（登录日志）

history （history -c 清理历史记录）这时按up就没有上一条命令的记录了

还要清理bash_history

网络配置

网卡配置 cat /etc/network/interfaces
dns配置 cat /etc/resolv.conf
查看主机名 hostname

启动网卡 ifup ens‘数字’

关闭网卡 ifdown ens‘数字’

vim /etc/network/interfaces

dhcp自动模式(不能获取静态ip）

static手动模式

重启cat/etc/network/interfaces查看

网络拓扑收集

日志信息收集

/var/log/*

http日志：/var/log/apache2/access.log

mysql日志：/var/log/mysql/error.log

apt日志：cat /var/log/apt
